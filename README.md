📊 KinKeeper 🚀

KinKeeper is a modern relationship management web app that helps users stay connected by tracking and managing interactions like calls, texts, and video chats, ensuring no important relationship is overlooked.

🛠️ Technologies Used
⚛️ React.js
🔀 React Router DOM
🎨 Tailwind CSS + Component Library
📈 Recharts

✨ Key Features

📅 Smart Timeline
Track and organize all interactions in a clean, structured, and filterable timeline.

🎯 Interaction Analytics
Visualize communication patterns with dynamic charts for better insights.

🔍 Advanced Filtering
Filter interactions by type (Call, Text, Video) to quickly find specific activities.
